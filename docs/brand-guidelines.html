02 / Typography
Three Voices
A display serif for presence, a modern sans for clarity, and a monospace for data. Each has a job. None overlap.
Display — DM Serif Display
Pangea.
One Platform. Every Worker. Site Ready.
Use for: brand name, hero headings, section titles. Never for body text or UI labels.
Body — DM Sans
Site Ready automates everything between finding a construction worker and getting them on site — cert verified, right-to-work checked, H&S signed off — all on their phone, no paperwork. Workers move from application to site-ready through a single messaging thread.
Use for: body text, buttons, navigation, form labels. Weights 300–700. Primary workhorse.
Mono — JetBrains Mono
CSCS-BLUE-2847  |  EXPIRY: 2027-03-15  |  STATUS: VERIFIED  |  SITE: NORTHERN-014
Use for: data values, status labels, card numbers, timestamps, code, technical references.

04 / Usage Rules
Do & Don't
Do

Use Forest-800 (#1B4332) as the primary brand colour for navigation, headers, and primary actions.

Don't

Use forest green for error states or destructive actions. Red is always danger. Green is always safe/verified.

Do

Use Copper-500 (#C17F59) sparingly for CTAs, accent highlights, and attention-drawing elements.

Don't

Use copper as a background fill or for large surface areas. It's an accent, not a primary.

Do

Use DM Serif Display only for the brand name and major section headings. Keep it special.

Don't

Use the display serif for buttons, form labels, table headers, or body text. That's DM Sans territory.

Do

Use monospace (JetBrains Mono) for all data: CSCS numbers, dates, statuses, worker IDs, timestamps.

Don't

Mix data fonts. If a value is a number, date, or status code, it's always monospace. No exceptions.

05 / Voice & Tone
How Pangea Speaks
The Voice

Direct. Construction people don't have time for waffle. Say what it does. Say what it costs. Say how long it takes. Done.

The Voice

Grounded. No jargon. No "leverage synergies." Workers use this product. Speak like a site manager, not a SaaS marketer.

Example

Good: "Worker verified. CSCS valid until March 2027. Ready for site."
Bad: "Congratulations! Your onboarding journey is complete. Welcome aboard!"

Example

Good: "3 cards expire this month. View workers."
Bad: "Action required: Please review your upcoming certification renewals dashboard."
